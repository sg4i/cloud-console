# Design

## 功能

- 调用云服务凭据扮演角色登录控制台
- 支持腾讯云、阿里云、aws等云服务提供商
- 程序支持命令行模式、grpc微服务模式、lib库模式
- 配置支持环境变量方式和配置文件的方式
- 记录云API调用链，按时间序列，基于云审计日志分析防御
